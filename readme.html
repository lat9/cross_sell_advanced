What it does

This Zen Cart plugin enables a site to "cross-sell" products. An admin tool is provided that is used to 'connect' the various cross-sells for the site and a simple storefront 'centerbox' is provided for the cross-sells' display. Additional documentation can be found on the plugin's GitHub repository.

Installation

There are no core-file overwrites in this plugin, but you should always backup your cart's database and files prior to making any changes.

Copy the following files to your cart's installation, after renaming the "YOUR_ADMIN" directory to match your custom admin's name:

  1. /includes/extra_datafiles/xsell_definitions.php
  2. /includes/languages/english/extra_definitions/xsell_products_definitions.php
  3. /includes/modules/xsell2_products.php
  4. /includes/templates/bootstrap/templates/tpl_modules_xsell2_products.php
  5. /includes/templates/template_default/templates/tpl_modules_xsell2_products.php
  6. /YOUR_ADMIN/xsell.php
  7. /YOUR_ADMIN/includes/auto_loaders/config.xsell.php
  8. /YOUR_ADMIN/includes/classes/dbio/DbIoProductsXsellHandler.php
  9. /YOUR_ADMIN/includes/classes/observers/XsellAdminObserver.php
  10. /YOUR_ADMIN/includes/extra_datafiles/xsell_advanced_definitions.php (No longer used, can be safely removed)
  11. /YOUR_ADMIN/includes/functions/extra_functions/cross_sell_plus_advanced_page_registration.php (No longer used, can be safely removed)
  12. /YOUR_ADMIN/includes/functions/extra_functions/zen_cfg_read_only.php
  13. /YOUR_ADMIN/includes/init_includes/init_xsell_config.php
  14. /YOUR_ADMIN/includes/javascript/xsell.php
  15. /YOUR_ADMIN/includes/languages/english/dbio/DbIoProductsXsellHandler.php
  16. /YOUR_ADMIN/includes/languages/english/extra_definitions/advanced_xsell_defs.php (No longer used, can be safely removed)
  17. /YOUR_ADMIN/includes/languages/english/extra_definitions/x-sell_defs.php
  18. /YOUR_ADMIN/includes/modules/xsell/category_product_selection.php

Now that the base files have been copied, go to your site's admin and refresh the page to cause the plugin's installation script to run. Next step is to add the cross-sell information to your template's tpl_*.info_display.php files. See this online documentation for the process.

Note: If your site is using the ZCA Bootstrap Template, don't forget to copy the file /includes/templates/bootstrap/templates/tpl_modules_xsell2_products.php to your "clone" of the template!

Uninstall

First, delete the files that were copied to your zen-cart installation's admin directories, then use your admin's Tools->Run SQL Patches to run the file xsell_uninstall.sql.
